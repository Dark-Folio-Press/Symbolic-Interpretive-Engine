# Prompt Templates for Symbolic Analysis

These templates are designed for deeper symbolic and archetypal analysis beyond straightforward chart interpretation. Use these when exploring patterns, themes, and symbolic relationships.

## Archetypal Pattern Analysis

### Template: Identifying Archetypal Themes

```
Analyze the following chart configuration for archetypal patterns:

Configuration:
[Describe the planets, signs, houses, and aspects involved]

Task: Identify and explore the dominant archetypal themes present in this configuration.

Please:
1. Name the primary archetypes at play (e.g., Hero, Caregiver, Rebel, Sage)
2. Explain how these archetypes manifest through the specific planetary energies
3. Discuss the relationships and tensions between archetypes
4. Consider how these themes might express across different life domains
5. Explore both light and shadow expressions of each archetype
6. Suggest paths for integration and wholeness

Framework: [Jungian/Hillman/Mythological/Other]
Depth: [Overview/Moderate/Deep dive]
```

### Example Usage

```
Analyze the following chart configuration for archetypal patterns:

Configuration:
- Mars in Aries in the 1st house conjunct Chiron
- Sun in Cancer in the 4th house
- Moon in Capricorn in the 10th house opposite Sun

Task: Identify and explore the dominant archetypal themes present in this configuration.

Please:
1. Name the primary archetypes at play
2. Explain how these archetypes manifest through the specific planetary energies
3. Discuss the relationships and tensions between archetypes
4. Consider how these themes might express across different life domains
5. Explore both light and shadow expressions of each archetype
6. Suggest paths for integration and wholeness

Framework: Jungian with focus on the wounded healer and individuation
Depth: Deep dive
```

## Declination and Out-of-Bounds Analysis

**Note:** Declination refers to a planet's distance north or south of the celestial equator (measured in degrees). Parallels occur when two planets have the same declination; contra-parallels occur when they have equal but opposite declinations. Planets are considered "out-of-bounds" when they exceed approximately 23Â°27' declination.

### Template: Parallel and Out-of-Bounds Study

```
Analyze the following declination factors:

Parallels:
- [Planet 1] parallel [Planet 2] at [degrees]

Contra-parallels:
- [Planet 1] contra-parallel [Planet 2] at [degrees]

Out-of-bounds planets:
- [Planet] at [degrees] declination

Please explore:
1. How parallels function similarly to conjunctions
2. How contra-parallels function similarly to oppositions  
3. The special significance of out-of-bounds planets
4. How these factors add dimension to the chart interpretation
5. Integration with traditional aspects
6. Notable manifestations of these energies

Note: These factors add depth but should enhance, not replace, traditional analysis.
```

## Using These Analysis Templates

1. **Select the appropriate template** for your analytical focus
2. **Gather necessary chart data** before starting
3. **Fill in all required fields** with specific information
4. **Set clear intentions** for depth and approach
5. **Allow space for emergence** - symbolic analysis can reveal unexpected patterns
6. **Document insights** for future reference
7. **Share useful discoveries** with the community

## Combining Templates

These templates can be combined for multi-dimensional analysis:
- Archetypal themes + Mythological amplification
- Elemental analysis + Shadow work
- Cycle analysis + Transit forecast
- Chart essence + Relationship patterns

## Best Practices

- Start with broader patterns before detailed analysis
- Let symbols speak before imposing interpretation
- Hold multiple perspectives simultaneously
- Honor mystery and ambiguity
- Document methodology for reproducibility
- Share findings while respecting privacy

---

*Symbolic analysis is both art and science. These templates provide structure while leaving room for intuition and insight.*
